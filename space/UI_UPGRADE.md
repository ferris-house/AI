## 🎨 色彩方案

### 主题色
| 颜色 | 色值 | 用途 |
|------|------|------|
| 青色 | #0ff | 主色调、边框、发光 |
| 品红 | #f0f | 次色调、健康条、底部 |
| 黄色 | #ff0 | 强调色、combo、警告 |
| 绿色 | #0f0 | 成功、正确率、简单 |
| 红色 | #f00 | 危险、错误、困难 |

### 渐变方案
```css
/* 青紫渐变 */
linear-gradient(135deg, #0ff, #f0f)

/* 深色渐变 */
linear-gradient(135deg, rgba(0,0,0,0.95), rgba(0,50,50,0.95))

/* 难度渐变 */
Easy:   linear-gradient(135deg, #0f0, #0c0)
Medium: linear-gradient(135deg, #ff0, #fc0)
Hard:   linear-gradient(135deg, #f00, #c00)
```

## 📝 使用建议

### 性能模式
如果玩家电脑性能较弱，可以：
1. 减少 box-shadow 层数
2. 降低 blur 数值
3. 禁用 backdrop-filter
4. 减少动画数量

### 自定义主题
修改主题色：
```css
/* 找到这些CSS变量并修改 */
#0ff → 你的主色
#f0f → 你的次色
#ff0 → 你的强调色
```
