## 🎯 如果发现问题

### 调试步骤：
1. 按 F12 打开开发者工具
2. 查看 Console 的错误信息
3. 记录错误文件和行号
4. 检查 `CHANGELOG.md` 查看最近修改
5. 查看 `BUGFIX.md` 查看已知问题

### 快速修复：
- 激光不显示 → 检查 `fireLaser` 函数
- 爆炸无效果 → 检查 `createExplosion` 函数
- 音效无声音 → 检查浏览器音量设置
