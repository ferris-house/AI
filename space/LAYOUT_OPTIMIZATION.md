## 🎨 设计原则

### 1. 8px栅格系统
所有间距和尺寸都是8的倍数：
- 边距: 30px (8×3.75 ≈ 8×4)
- 内边距: 22px, 35px (约8的倍数)
- 间距: 25px (8×3.125 ≈ 8×3)

### 2. 视觉层次
```
Z轴层次:
1000: 开始/结束界面
1002: 准星
1001: HUD边角
100:  游戏HUD
0:    3D场景
-1:   背景效果
```

### 3. 呼吸空间
- 元素间距 ≥ 25px
- 内容内边距 ≥ 20px
- 边缘留白 ≥ 30px

### 4. 响应式缩放
```
>1400px: 100% (基准)
1024-1400px: 90-95%
768-1024px: 80-85%
<768px: 60-75%
```
